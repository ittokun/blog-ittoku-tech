# ittoku-tech

このリポジトリには、ドメイン名`ittoku-tech.com`のソースコードが置かれています。

以下のコマンドを実行することで、動作を確認することができます。

試してみましょう！

### 動作要件

ここにあるソースコードを実行するためにはDockerが必要になります。

インストールしておきましょう。

### 動かす

アプリの起動

```bash
make run
```

アプリの終了

```bash
make down
```

アプリにアクセス

```bash
make web-bash
```

ESlintを実行

```bash
make web-lint
```

### ページを見る

以下のURLにアクセスすることで、Webページを閲覧することができます。

```bash
curl http://0.0.0.0:3000
```
